# ai-calorie-tracker# AI Calorie Tracker

Simple nutrition tracking web app.

Users enter food name and the app calculates calories,
protein, carbs and fat using the Nutritionix API.

## Tech Stack

HTML  
CSS  
JavaScript  
Nutritionix API

## Features

Food nutrition lookup  
Calories calculation  
Protein / carbs / fat tracking

## Future Improvements

Barcode scanning  
Daily calorie tracker
